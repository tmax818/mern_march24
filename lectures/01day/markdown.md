### Variables


```javascript
var name = "Tyler"
```


```javascript
let name = "Tyler"
const FIRST_NAME = "Tyler"
```


How do you know when to use `let` or `const`?

Use `const` unless you have to use `let`!
<!-- .element: class="fragment"  -->



## Variable Scope


local or global


```javascript

let global = "I'm global!"

const func = () => {
    let local = "I'm local!"
}

```
If it's not in a function, it's global!!
